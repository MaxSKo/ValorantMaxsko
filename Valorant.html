Règles du jeu

Le “tactical shooter” : info, économie, objectifs.

Objectif

Attaque : poser le Spike et le défendre. Défense : empêcher la pose ou désamorcer.

Rounds

Alternance attaque/défense. Gérer le chrono, les rotations et l’info.

Économie

Achats d’armes/armure/compétences. Le “force” (Acheter malgré la pauvereté), “eco” (Economisé pour le prochain round), “full buy” (Vandal/Phantom + Gros shield) font la diff.

  • Conditions pour la victoireSpike explosé / désamorcé / éliminations / temps
  • Infosons, utilitaires, “contact”, timings, fake
  • Tradejouer à 2 : prendre un duel et échanger

Modes

Pour chill, s’entraîner ou tryhard.

Non classé

Le mode “classique” sans impact sur ton rang. Parfait pour apprendre le jeu/Gérer la pression d'une game longue (13 round gagnant sans prolongation)

Classé

Game classé : Rank + MMR caché, tryhard constant pour devenir le meilleur, se finit en 13 round gagnant hors prolongation

Deathmatch

Aim + routines. Priorité : crosshair placement, strafes, discipline.

  • Swiftplay / Spike RushGame rapide (5 ou 4 round gagnant sans prolongation), rapide, échauffement
  • CustomEntrainement 1v1, Stratégies, Entrainement Lineup

Agents

Rôles & idées de picks.

Duelist

Créer l’espace : dash/entry, prendre le premier contact, casser les setups.

Initiator

Gagner l’info : flashes / reveals / drones pour aider l’entry.

Controller

Fumigènes : couper les lignes, contrôler la map, tempo en post-plant.

Sentinel

Tenir un site : anti-flank, contrôle de zone, pièges, retake plus safe.

Composition
  • 1 Controllerobligatoire sur la plupart des maps
  • 1 Initiatorpour l’info + flashes
  • 1 Sentinelanti-flank + tenue de site
  • 1–2 Duelistsselon style (double entry ou pas)

Maps

Ce bloc est fait pour tes résumés + callouts.

Plan de lecture

1) Control du mid → 2) Side de pression → 3) split / exec.

Défense

Info + retake. Ne pas donner 1v1. Jouer la montre et l’ulti.

Attaque

Utilitaires “en chaîne” : info → flash → smoke → entry.

Exemple

Bind : Attaque rapide, prise A : 2 shower, 2 short, 1 lurk (sentinel) Défense : 2 A, 2 B, 1 Spawn (smoker de pref)

Tips & Compo

Des règles simples qui font monter vite.

Routine aim

10–15 min : DeathMatch focus constant/ Crosshair placement + micro-strafes + discipline.

Comms

Court & utile : “2 main”, “Flash droite”, “rotate”, “Go retake.

Post-plant

Jouer crossfire + ne pas peek solo. Tempo sur le tap + spell.
